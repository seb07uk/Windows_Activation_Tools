Windows Activation & License Toolkit

A multilingual Windows licensing management tool (EN/PL)

This repository contains an advanced, interactive Batch script designed to simplify Windows license management. The toolkit is ideal for IT administrators, technicians, and power users who need quick access to activation status, license details, OEM keys, and essential licensing operations.

The script supports English and Polish, automatically checks for administrator privileges, and uses ANSI color formatting for a clean, readable interface.

Features

  • Display activation status
  • Basic license information
  • Detailed license information
  • Read OEM key from BIOS/UEFI
  • Restart the sppsvc licensing service
  • Install a new product key
  • Activate Windows

Installation & Usage

  1. Download the file Windows Activation Tools (en-pl).bat
  2. Run it as Administrator
  3. Select your language
  4. Use the menu to perform licensing operations

Requirements

  • Windows 10 or Windows 11
  • Administrator privileges
  • ANSI support (enabled automatically)

Legal Notice

This tool does not provide product keys and does not bypass activation mechanisms.
All operations rely on official Windows commands (slmgr, wmic, net).
Use only in accordance with Microsoft licensing terms.
